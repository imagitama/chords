# ChordMaster

A React app that shows chords, keys and more.

## Usage

    npm i
    npm start

## About

Technologies:

- create-react-app (not ejected)
- Redux
- Emotion (with theming)

All chords are defined in `src/chords.js` and all keys in `src/keys.js`.
