# Chord Guide

A React web app that lists all popular and not popular chords, keys and songs.

## Usage

    npm i
    npm start

## About

Technologies:

- create-react-app (not ejected)
- Redux
- Emotion (with theming)
- ReactRouter

All chords are defined in `src/chords.js` and all keys in `src/keys.js`.
